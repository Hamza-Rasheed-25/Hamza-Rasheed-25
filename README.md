### 🚀 The Mission

I am a Full-Stack Software Engineer and a known, trusted Community Builder based in Lahore. My work revolves around breaking down barriers, making it easier for others to contribute, and mastering new technologies. 

I believe that **"Open-Source Mindset", "Transparency", "Education", and "Collaboration"** are what both the software and hardware communities desperately need. My ultimate goal is to bridge the gap and create resources that lower the learning curve, ensuring that every user feels as confident and capable with their devices as a developer feels when their code works on the first try.

### 📦 Featured Open-Source Projects

🕌 **[Azaan CLI](https://github.com/Hamza-Rasheed-25/azaan-cli)** — *A lightweight, beautiful, and interactive CLI to check Azaan times anywhere in the world right from your terminal.*
* **Features:** Interactive setup, status bar string outputs for custom environments (like tmux or Waybar), and a zero-setup background daemon that sends native desktop notifications 10 minutes before and exactly at prayer time.
* **Tech Stack:** Built with Node.js, Commander.js, Clack Prompts, Axios, and Node-Notifier.
* **Try it out:** `npm install -g azaan-cli`
